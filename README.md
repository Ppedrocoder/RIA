# Projeto RIA - Ateliê Potiguar

## Conceito
O projeto se baseia na ideia de criar um ambiente virtual para artistas potiguares, nascidos no Rio Grande do Norte,
terem suas obras expostas, gerando um acervo de fácil consumo baseado na cultura potiguar.

## Objetivo
Valorizar a cultura potiguar, tornando visível tanto para moradores locais como de fora a singularidade de uma cultura que é omitida em meio a diversas de um país continental.

## Tecnologias
- Angular
- Git e Github
